## 2. Local processing of user content

Text, series metadata, episodes, sources, reflection notes, and selected images
are processed locally in the browser. The application has no account system,
database, or server-side storage for this user content.

PDF output and `.seriescreator` project backups are generated locally. Backups
are created or read only on the device and are not sent to the operator.

## 3. Local storage

The application uses `localStorage` to store the selected language and current
draft on the device. The application does not send these values to the operator.
They can be removed through browser controls or through **New / Clear**.
